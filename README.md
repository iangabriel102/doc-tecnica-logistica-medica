# Controle de Suprimentos e Medicamentos - Documentação Técnica

Este repositório contém uma documentação técnica de 5 páginas estruturada em **LaTeX**, focada na gestão logística de insumos médicos e fármacos.

## 📋 Funcionalidades do Documento
- **Gestão de Inventário:** Metodologia de catalogação de entrada de suprimentos.
- **Controle de Validade:** Implementação do sistema PVPS (Primeiro que Vence, Primeiro que Sai).
- **Rastreabilidade:** Tabelas detalhadas com data de chegada, quantidade e status de vencimento (Seguro, Alerta, Crítico).
- **Conformidade:** Baseado nas normas de segregação de itens vencidos (RDC 222/2018 da ANVISA).

## 🛠️ Tecnologias Utilizadas
- **LaTeX** (Editor Overleaf)
- **Pacotes:** `tabularx` (tabelas dinâmicas), `xcolor` (sinalização de status), `fancyhdr` (formatação técnica).

## 📂 Arquivos no Repositório
- `main.tex`: Código-fonte do documento.
- `documento_final.pdf`: Versão compilada para leitura direta.
